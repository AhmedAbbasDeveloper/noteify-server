# Noteify Server

This repo contains the server for [Noteify](https://noteify-app.netlify.app), a distributed note taking app. The [front-end client](https://github.com/AhmedAbbasDeveloper/noteify-client) makes requests to this API so users can add and remove notes.

The project uses the following tech stack:
- TypeScript
- GraphQL/Apollo
- TypeORM & PostgreSQL
- Heroku
